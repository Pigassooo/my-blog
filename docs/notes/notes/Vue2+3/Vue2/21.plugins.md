# 21. Plugins

### 21.1 介绍

插件是 Vue.js 的一个重要概念，它可以用来扩展它的功能。插件可以包含全局方法、全局属性、组件、指令等。

插件的使用可以帮助我们解决一些重复性的工作，比如表单验证、日期格式化、Ajax 请求等。
